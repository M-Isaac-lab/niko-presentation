# Image Dashboard

Pour afficher l'image du dashboard dans la présentation :

1. Placez votre capture d'écran du dashboard dans ce dossier (`/public`)
2. Nommez le fichier : `dashboard-screenshot.png`
3. Format recommandé : PNG ou JPG
4. Dimensions recommandées : 1920x1080px (16:9) pour un meilleur rendu

L'image sera automatiquement affichée dans la slide "Notre Solution".
